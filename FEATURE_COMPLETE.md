## 🔧 技术特性

### 核心技术栈
- **前端**: React 18 + TypeScript + Vite
- **UI组件**: shadcn-ui + Tailwind CSS
- **后端**: Supabase (数据库 + 认证 + 实时功能)
- **图片存储**: 免费图床服务 (ImgBB)
- **状态管理**: React Context + React Query
